ABOUT

This is an simple, user-friendly system which can be used to store information about the students and their marks and attendance. The details updated by the staffs can be viewed by students using login credentials.

Features

Student Information Management System is a powerful student management software comes with an incredible set of features to assist with your class administration tasks and complex processes.

Student Information

Get easy access to student particulars, department, date of birth, blood group and their marks etc.. anytime.

Attendance

Student attendance in each subject can also be viewed in this system.

Gradebook

The online gradebook feature enables teachers to enter grades frequently.

Report Card

The marks obtained by students will be available online once the teacher updates it.

Sign-In & Sign-Out

Students and Teachers can sign in and sign out seamlessly using their User-Id and password generated by the system

Secure

Data can be stored in a secure way and can be accessed anytime by students.
